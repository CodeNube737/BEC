# BEC - Sand Dune Searcher

Arduino Uno R3 motor controller for a 4-wheel robot with H-bridge control.

## Quick Start
1. See [ARDUINO_README.md](ARDUINO_README.md) for setup and usage instructions
2. See [WIRING_GUIDE.md](WIRING_GUIDE.md) for hardware connections
3. Upload `motor_controller.ino` to your Arduino Uno R3

## Features
- Digital input control (forward/backward/left/right)
- Smooth speed ramping (acceleration/deceleration)
- H-bridge motor driver support
- Controls 2 sets of 2 wheels (4 wheels total)
